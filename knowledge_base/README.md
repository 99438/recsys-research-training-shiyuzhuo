# 知识库索引

本目录用于积累科研训练过程中的基础知识、概念理解和工具使用经验。

## 文件说明

- **`python.md`**：Python 编程基础笔记，包括 pandas 数据处理、代码片段等
- **`math.md`**：机器学习/深度学习所需的数学基础笔记
- **`recsys.md`**：推荐系统相关概念、术语、模型笔记
- **`paper_reading.md`**：论文阅读笔记模板与具体论文的精读记录
- **`tools.md`**：工具使用笔记（Git、SSH、服务器、Zotero 等）

## 使用说明

- 遇到不懂的概念时，先查询后记录到对应文件
- 重要代码片段、命令和配置保存到相应笔记中
- 定期回顾和整理，合并重复内容
- 建立文件内部链接，方便知识关联
