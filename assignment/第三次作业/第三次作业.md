## 七、遇到的问题及解决

### 问题1：PyTorch 2.12.1不支持CUDA 12.1

**现象：** 
```
ERROR: Could not find a version that satisfies the requirement torch==2.12.1
```

**原因：** 
课程推荐的PyTorch 2.12.1只有CUDA 12.6版本，我的驱动只支持CUDA 12.3

**排查过程：**
1. 运行 `nvidia-smi` 查看驱动CUDA支持版本 → 12.3
2. 查看PyTorch官方cu121版本可用范围 → 最高2.5.1
3. 选择 PyTorch 2.5.1 + CUDA 12.1（兼容12.3驱动）

**解决方法：**
```powershell
# 使用CUDA 12.1版本的PyTorch 2.5.1
& "D:\anaconda\envs\lesson03\python.exe" -m pip install torch==2.5.1 --index-url https://download.pytorch.org/whl/cu121
```

**结果：** 安装成功，GPU可用，实验正常运行

### 问题2：Conda activate 在PowerShell中不生效

**现象：** 运行 `conda activate lesson03` 后，`python --version` 仍显示 base 环境的 Python 3.13.9

**原因：** PowerShell中conda未正确初始化

**解决方法：** 使用完整路径直接调用环境的Python
```powershell
& "D:\anaconda\envs\lesson03\python.exe" [命令]
```

### 问题3：NumPy警告
**现象：** 每次运行都出现 "Failed to initialize NumPy" 警告

**影响：** 不影响PyTorch核心功能，SASRec正常运行

**说明：** 这是PyTorch的可选依赖，本课程不需要处理
