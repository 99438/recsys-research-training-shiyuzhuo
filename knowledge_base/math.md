### 微积分

- [x] 导数与梯度
  - 导数：函数在某点的变化率
  - 梯度：多元函数对各变量的偏导数向量
  - 梯度指向函数增长最快的方向
- [x] 梯度下降
  - 参数更新公式：`θ = θ - lr * ∇L(θ)`
  - `lr`：学习率，控制更新步长
  - 目标：沿梯度反方向移动，使损失函数下降
- [x] 反向传播
  - 从输出Loss开始，逐层计算梯度
  - PyTorch中通过`loss.backward()`自动完成
  - 链式法则保证了梯度正确传递
- [ ] 链式法则（数学推导）

### 优化理论

- [x] 损失函数（Loss Function）
  - 衡量模型预测与真实标签的差距
  - **二分类交叉熵**（BCE）：`-[y*log(p) + (1-y)*log(1-p)]`
  - PyTorch中使用`BCEWithLogitsLoss`（输入logit，内部自动sigmoid）
  - Loss必须是标量，才能进行反向传播
- [x] 优化算法
  - **SGD**（随机梯度下降）：`θ = θ - lr * grad`
  - **Adam**：自适应学习率，结合动量和RMSProp
  - 常用超参数：学习率（lr）、动量（momentum）

- [ ] 正则化（L1/L2）

## 常用公式

### Sigmoid函数

```
σ(x) = 1 / (1 + e^(-x))
```

- 将实数映射到(0, 1)区间
- 用于二分类，将logit转为概率

### 二分类交叉熵损失（BCE Loss）

```
L = -[y * log(σ(z)) + (1-y) * log(1-σ(z))]
```

- `y`：真实标签（0或1）
- `z`：模型输出的logit
- `σ(z)`：预测概率

### 梯度下降更新

```
θ_{t+1} = θ_t - η * ∇L(θ_t)
```

- `θ`：模型参数
- `η`：学习率
- `∇L`：损失函数对参数的梯度

### 点积（内积）

```
u · v = u₁v₁ + u₂v₂ + ... + uₙvₙ = Σ(uᵢvᵢ)
```

- 用于计算向量相似度
- 推荐系统中：`score(u,i) = e_u^T · e_i`
