# 个人学习计划

## 培训阶段目标（2026.7 - 2026.8）

### 总体目标
完成一次最小科研闭环，产出：
- 论文阅读笔记
- 代码运行日志
- 实验环境与依赖记录
- 实验结果截图或表格
- 问题排查记录
- 阶段汇报材料

### 基础能力补齐重点

根据自身基础情况，本次培训重点关注：

**已掌握**
- ✓ Python 编程基础
- ✓ 深度学习基础经验
- ✓ Git/GitHub 使用

**环境配置**（重点补齐）
- conda 环境管理
- SSH 与服务器基本使用
- AutoDL、NAS 等服务器平台使用
- 日志查看、报错记录与问题排查

**推荐系统基础**（重点补齐）
- 用户、物品、交互数据
- 召回、排序与评价指标
- 推荐模型训练框架
- 推荐系统实验范式

**科研能力**（重点提升）
- 论文精读与方法拆解
- 代码精读与实验复现
- 结果分析与汇报表达
- 科研记录与问题追踪

### 学习方法

- 讲解为辅，自学为主
- 使用 Obsidian 保留代码、笔记、日志、截图和问题记录
- 使用 GitHub 进行版本管理和作业提交
- 定期汇报进展与遇到的问题
- 合理使用 AI 工具辅助学习（但保持独立思考和核查）
