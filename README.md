## 仓库说明

本仓库用于记录 2026 年暑期推荐系统科研入门培训的学习过程，使用 **Obsidian + Git** 进行知识管理和版本控制。

**记录内容：**
- 每日学习记录与进展
- 课后作业提交
- 论文阅读笔记
- 代码复现过程
- 个人知识库积累

**工具栈：**
- 笔记管理：Obsidian
- 版本控制：Git + GitHub
- 文献管理：Zotero（后续学习）

## 培训目标

完成一次最小科研闭环：
- 找到一篇适合入门的论文
- 读懂论文的基本问题、方法和实验
- 找到并运行相关代码
- 复现一个可检查的实验结果
- 记录过程、问题和分析
- 完成一次阶段性汇报表达
