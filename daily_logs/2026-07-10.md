## Git 仓库初始化完整流程

### 第一步：初始化本地仓库

```bash
# 进入项目目录
cd D:\research-notes\recsys-research-training-shiyuzhuo

# 初始化 Git 仓库
git init

# 查看当前状态
git status
```

### 第二步：添加文件并提交

```bash
# 添加所有文件到暂存区
git add .

# 查看暂存区状态
git status

# 提交到本地仓库
git commit -m "初始化仓库：建立目录结构和基础笔记"
```

### 第三步：创建 GitHub 远程仓库

1. 登录 GitHub 网站
2. 点击右上角 "+" → "New repository"
3. 仓库名填写：`recsys-research-training-shiyuzhuo`
4. 选择 Public（公开）
5. 不要勾选 "Initialize with README"（本地已有）
6. 点击 "Create repository"

### 第四步：关联远程仓库并推送

```bash
# 添加远程仓库
git remote add origin https://github.com/99438/recsys-research-training-shiyuzhuo.git

# 查看远程仓库信息
git remote -v

# 推送到远程仓库（首次推送需要 -u 参数）
git push -u origin main

# 如果提示分支名是 master，改用：
# git branch -M main
# git push -u origin main
```

### 后续日常更新流程

```bash
# 1. 查看修改状态
git status

# 2. 添加修改的文件
git add .
# 或添加指定文件：git add <file_path>

# 3. 提交到本地
git commit -m "描述本次修改内容"

# 4. 推送到 GitHub
git push
```

### 常用 Git 命令

```bash
# 查看提交历史
git log --oneline

# 查看文件修改详情
git diff

# 撤销工作区修改（谨慎使用）
git checkout -- <file>

# 查看远程仓库信息
git remote -v

# 拉取远程更新
git pull
```
