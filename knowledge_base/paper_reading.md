# 论文阅读笔记

## 论文阅读方法

### 核心阅读问题

**问题定义**

- 这篇论文解决什么任务？
- 研究对象是什么（用户、物品、序列、图、文本）？

**动机与创新**

- 论文的动机是什么？已有方法有什么问题？
- 论文的创新点是什么？

**方法理解**

- 核心方法是什么？
- 涉及几个模块？分别解决哪些问题？
- 关键公式和技术细节

**实验设计**

- 使用了哪些数据集和指标？
- 做了哪些实验（主实验、消融实验、参数敏感性分析）？
- 实验是如何设计的？

**结果分析**

- 结果比哪些 baseline 好？好在哪里？
- 哪些模块起了作用？
- 是否适合复现或改进？

### 配套要求

- 读懂代码
- 能对应论文方法和代码实现
- 理解实验设置和结果产生过程

## 文献调研方法论

### 三类文献的调研顺序

1. **综述**：搭建领域整体地图，标题含 `survey` / `review` / `overview`
2. **代表性经典**：吃透核心问题与基础方法（如序列推荐里的 SASRec）
3. **近年最新**：追前沿、看现存争议（一般限定近 2–3 年）

### 三种论文载体 & 核验重点

| 载体            | 核验字段                           |
| --------------- | ---------------------------------- |
| 会议论文        | 会议名称、举办年份、正式论文集     |
| 期刊论文        | 期刊名、卷号期号、页码、年份、DOI  |
| 预印本（arXiv） | 版本号、提交日期、后续正式录用情况 |

**版本演进**：作者初稿/预印本 → 会议正式录用版 → 期刊扩展完整版

**重要认知**：arXiv 只是预印本平台，**不代表被录用**；引用必须使用正式版。

### 检索工具组合

| 工具           | 职责                              | 局限                             |
| -------------- | --------------------------------- | -------------------------------- |
| Google Scholar | 发现候选（全文检索、被引/相关）   | 全文检索可能过宽；PDF 常为预印本 |
| DBLP           | 核验规范书目、导出 BibTeX         | 只匹配标题，不做全文检索         |
| CCF Rank 插件  | 在 Scholar/DBLP 页面显示 CCF 分级 | 数据可能有延迟，以官方目录为准   |

**完整核验链路**：`Scholar 筛选 → DBLP 核对 → DOI/出版社官网确认正式版`

### 检索式构建

**通用组合公式**：`研究对象 × 学习机制 × 任务`

**标准句式模板**：

> 我想了解【XX 任务】中【XX 方法】如何解决【XX 问题】，同时限定数据集、评价指标、代表性论文

**迭代记录要求**（每轮）：

- 关键词增删替换的原因
- 结果状态：过宽 / 过窄 / 偏离主题
- 下一轮调整方案

### 引用关系追踪三维度

以种子论文为起点：

- **向后追溯**：参考文献（前序基础、经典方法、问题来源）
- **向前追溯**：被引文献（后续改进、复现、批判类工作）
- **横向拓展**：相关论文（同期同类方法、邻近任务、同作者系列）

**实例**：以 SASRec 为种子 → 向前找到 BERT4Rec / CL4SRec / ICLRec / DCRec / Recformer 一系列改进工作。

### 10 分钟快速筛选流程

| 分钟 | 看什么                             | 判断问题                       |
| ---- | ---------------------------------- | ------------------------------ |
| 1'   | 元数据（标题/作者/年份/载体/版本） | 论文身份、版本是否清晰可核验？ |
| 2'   | 摘要                               | 任务/方法/结论是什么？         |
| 3'   | 引言 + 结论                        | 相比现有方法创新点在哪？       |
| 2'   | 模型图 + 主实验表                  | 实验证据强不强？               |
| 2'   | 开源代码 / 数据集 / 项目主页       | 是否具备精读、复现价值？       |

**筛选结果三分类**：**精读 / 备查 / 排除**（"排除"仅指本次调研不匹配，不代表论文质量差）

**避坑**：

- 只有 arXiv 无正式发表 → DBLP/DOI 核验录用情况
- 只看分区、引用量 → 必须回到摘要、实验、匹配度

### 论文等级评价体系速记

| 体系         | 发布方             | 适用对象    | 使用边界                       |
| ------------ | ------------------ | ----------- | ------------------------------ |
| CCF 推荐目录 | 中国计算机学会     | 会议 + 期刊 | 载体等级 ≠ 单篇论文质量        |
| 中科院分区   | 中科院文献情报中心 | 期刊        | 大类/小类分区分开使用          |
| JCR 分区     | Clarivate 科睿唯安 | 期刊        | JIF + Q1–Q4，标注学科类别+年份 |

**关键禁忌**：

- 中科院一区 ≠ JCR Q1，不可混用
- 期刊分区规则**不适用于**会议 A/B/C 分级

### 推荐系统方向主流载体

| 研究方向           | 主要发表载体              |
| ------------------ | ------------------------- |
| 推荐系统、信息检索 | RecSys、SIGIR、CIKM、WSDM |
| Web 挖掘、通用 AI  | WWW、KDD、AAAI、IJCAI     |
| 通用机器学习顶会   | NeurIPS、ICML、ICLR       |
| 权威期刊           | TOIS、TKDE、TIST          |
